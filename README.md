# Roulette Wheel YOLOv7 Dataset Repository

This repository contains a dataset for training a model to recognize roulette wheels using YOLOv7.

The dataset is structured as follows:

- dataset/
    - images/
        - wheel1_001.jpg
        - wheel1_002.jpg
        - ...
    - labels/
        - wheel1_001.txt
        - wheel1_002.txt
        - ...
